# BloodLink – Blood Donation Management System

## Project Overview

BloodLink is a web-based blood donation management system developed to connect blood donors with patients and hospitals. The platform helps users easily register as blood donors and allows patients to search for donors based on blood group and availability. The system aims to make the blood donation process faster and more efficient during emergencies.

## Features

* Donor registration and login
* Search donors by blood group
* Blood request management
* Admin dashboard to manage donors and requests
* Simple and user-friendly interface

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite

## How to Run the Project

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the Django server

python manage.py runserver

4. Open the browser and go to

http://127.0.0.1:8000/
